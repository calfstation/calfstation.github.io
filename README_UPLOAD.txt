CALF STATION GitHub Pages v1

[업로드 방법]
1. 이 ZIP의 압축을 풉니다.
2. 현재 열어둔 GitHub Upload files 화면의 큰 사각형에 다음 4개를 함께 끌어놓습니다.
   - index.html
   - generator 폴더
   - patches 폴더
   - README_UPLOAD.txt
3. 아래 Commit changes 버튼을 누릅니다.
4. 1~3분 뒤 https://calfstation.github.io/ 를 새로고침합니다.

[패처 테스트]
- 기본 주소는 테스트 모드입니다. 원본 ROM과 IPS를 각각 직접 선택해서 패치 작동을 확인할 수 있습니다.

[실제 패치 배포]
1. https://calfstation.github.io/generator/ 에 접속합니다.
2. 클린 원본과 IPS를 선택하고 '검사 및 생성'을 누릅니다.
3. '난수명 패치 다운로드'로 받은 .bin 파일을 GitHub 저장소의 patches 폴더에 올립니다.
4. 생성된 티스토리 서식 코드와 패처 주소를 사용합니다.

[중요]
- 브라우저 패처이므로 패치 파일의 완전한 은닉은 불가능합니다.
- 원본 ROM은 GitHub나 티스토리에 업로드하지 마세요.
- 현재 v1은 IPS 전용입니다.
